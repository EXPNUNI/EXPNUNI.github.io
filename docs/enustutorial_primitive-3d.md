# 3D Primitive Object - 기초 객체

---

enuSpace에서 제공하는 3D 기초객체의 사용방법에 대하여 설명합니다.

| Primitive Object | Description |
| :--- | :--- |
| box(./enusappapi_enucreate3dbox.md)  | 박스 모양 객체 |
| cone | 원뿔 모양 객체 |
| cylinder | 원통형 객체 |
| sphere | 구 객체 |
| text | 텍스트 객체 |
| lineset | 라인 객체 |
| faceset | 면세 객체 |
| inline | 인라인 객체 |
| terrain | 지형 객체 |
| contour | 컨투어 객체 |
| scatter chart | 스케터 차트 객체 |
